# I've set up my account. Now what?

Welcome to Stardance! You've done the hardest part already- now all that's left is making something awesome!

## 1. Figure out what to make
To get started making a project on stardance, you have two options:
- You can either follow a [Mission](https://stardance.hackclub.com/missions) (a guided build with a set goal)
- or figure out what to make on your own with a custom project

Projects in stardance are either software (time logged through a code editor), or hardware (time logged through timelapses). When you're setting up your project, you'll be able to choose which you want to start with. Don't worry- you can always change it later!

## 2. Set up Hackatime
We use Hackatime to count the time you spend building. If you went through all the setup, you already have an account, so you'll just want to make sure you have the hackatime extension installed on any code editor you use.

New to Hackatime, or running into issues? Follow the [Hackatime guide](https://stardance.hackclub.com/resources/hackatime).

After you've started working on a project and it's showing up in [Hackatime](https://hackatime.hackclub.com), make sure you edit your project to be able to link the hackatime project to your stardance project. This allows you to log the time on stardance!

## 3. Build and post devlogs
As you build, post **devlogs**- short updates on what you've been working on. They're how you show your progress as you're building your project. We recommend posting one every few hours, as it'll help later on when your project is being rated. Plus, it just helps to show what you've been working on! Learn what makes a good one in the [devlogs guide](https://stardance.hackclub.com/resources/devlogs).

As you're working on your project, I also suggest making frequent commits to github, as that's another way to both show your progress, plus it gives a backup if something goes wrong (I've had it save me many times :))

## 4. Ship it
When your project is ready, you **ship** it. Shipping basically just means that anyone on the internet can experience your project. If you want more info, check out [What does shipping mean?](https://stardance.hackclub.com/resources/what_is_shipping), then get a checklist tailored to your project type in [How to ship](https://stardance.hackclub.com/resources/how_to_ship).

## 5. Rate other projects
Before your Stardust unlocks, you'll rate a few other people's projects, and they'll rate yours. Projects are rated on originality, technicality, usability, and storytelling, and the stardust you get is based on what other people think of your project.

## 6. Spend your Stardust
Once your Stardust comes through, spend it in the [shop](https://stardance.hackclub.com/shop) on prizes. Then start your next project and do it all again!
